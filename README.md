# 📊 Coaching Center Management Dashboard (Microsoft Excel)

## 📌 Project Overview 

This project is a real-world Microsoft Excel dashboard developed for a local coaching center to replace their manual paper-based student and fee management system.

The coaching center previously maintained student records and monthly fee payments using registers, making it difficult to monitor fee collection, active students, and class-wise performance.

This dashboard digitizes the complete workflow and provides interactive reports using Excel.

---

## 🎯 Business Problem

The coaching center faced several challenges:

- Student information maintained in paper registers
- Manual monthly fee tracking
- Difficulty identifying students with pending fees
- No centralized report for monthly collection
- Time-consuming manual calculations
- No visual insights into class-wise performance

---

## 💡 Solution

Designed a complete Excel-based management system consisting of:

- Student Master Database
- Monthly Fee Tracker
- Interactive Dashboard
- Dynamic KPIs
- Pivot Tables & Pivot Charts
- Month Slicer for interactive reporting

The dashboard automatically updates after refreshing the Pivot Tables whenever new student or fee data is added.
